# Agent instruction files

This directory contains instruction files for configuring and guiding the behavior of AI agents.
Each file typically includes specific guidelines, rules, or parameters that the agent should
follow during its interactions.

## To setup in a new repo

These instructions should stay centralized in a single place.  In order
to use in a project, symlink them into the project directory.

For example:

```bash
cd ~/khan/frontend
ln -s ~/me/claude/CLAUDE.local.md CLAUDE.local.md
```
